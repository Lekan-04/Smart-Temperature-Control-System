# 🔥 Smart Temperature Control System 🌡️

## 📌 Overview

This project is a Smart Automation System built using Arduino UNO and LM35 temperature sensor.
It automatically activates LED and buzzer when temperature crosses threshold.

---

## ⚙️ Components Used

* Arduino UNO
* LM35 Sensor
* LED
* Buzzer
* 220Ω Resistor

---

## 🔁 Working

* LM35 reads temperature
* Arduino converts into Celsius

IF temperature > 30°C
👉 LED + Buzzer ON

ELSE
👉 OFF
